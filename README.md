## Overview

**JeepniGo** helps commuters track jeepney arrival times and seat availability in real-time, reducing waiting time and improving the daily commute experience.

### ✨ Enhanced Features

#### For Commuters:
- 🎫 **Smart Reservation System** - **NEW!** Book seats with full modal interface
- 📍 **Live Tracking View** - **NEW!** Real-time countdown timer and location
- 📜 **Trip History** - **NEW!** Review past rides, rate drivers, download receipts
- ⭐ **Favorites Management** - **NEW!** Quick book from saved routes
- ⏱️ **Real-time tracking** - See jeepney arrivals live
- 🎯 **"Sakay o Antay?" Decision Helper** - Smart recommendations on whether to board or wait
- 💺 **Seat Availability** - Color-coded status (plenty/few/full)
- 🇵🇭 **Bilingual Support** - Full Tagalog/English interface with local terminology
- ☔ **Weather Advisory** - Delay warnings for rainy conditions
- 📱 **QR Code Access** - Scan at stops for instant route status
- 📶 **Data-Saver Mode** - Ultra-light for 2G/3G networks

#### For Drivers:
- 🎤 **Voice Commands** - Hands-free status updates for safety
- 🗺️ **Live Location Sharing** - Real-time GPS broadcasting
- 💺 **Quick Seat Management** - Large touch-friendly + / - buttons
- 📊 **Trip Dashboard** - Daily summary and analytics
- 🔔 **Smart Notifications** - Bilingual alerts and confirmations
